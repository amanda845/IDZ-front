🚀 DÉPLOIEMENT EN 5 ÉTAPES

ÉTAPE 1 : INSTALLER LES DÉPENDANCES
------------------------------------
Ouvrez un terminal dans ce dossier et tapez :

    npm install

Attendez que ça se termine (1-2 minutes).


ÉTAPE 2 : CONFIGURER SUPABASE
------------------------------
1. Allez sur https://supabase.com → Créez un nouveau projet "autoluxe"
2. Attendez 2 minutes que le projet se crée
3. Allez dans "SQL Editor" → "New query"
4. Ouvrez le fichier `supabase/schema.sql` de ce dossier
5. Copiez TOUT le contenu et collez-le dans l'éditeur SQL
6. Cliquez "Run" (bouton vert)
7. Répétez avec le fichier `supabase/storage.sql`

✅ Vous devriez maintenant avoir 3 tables et 12 voitures dans votre base !


ÉTAPE 3 : CRÉER VOTRE FICHIER .env
-----------------------------------
1. Dans Supabase → Project Settings → API
2. Copiez "Project URL" et "anon public key"
3. Créez un fichier nommé exactement `.env` (avec le point devant)
4. Collez ceci dedans (remplacez par VOS valeurs) :

    VITE_SUPABASE_URL=https://VOTRE-PROJET.supabase.co
    VITE_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

5. Sauvegardez le fichier


ÉTAPE 4 : TESTER EN LOCAL (optionnel mais recommandé)
------------------------------------------------------
Dans le terminal :

    npm run dev

Ouvrez http://localhost:3000 dans votre navigateur.
Testez l'inscription, la connexion, la réservation.

Pour arrêter : Ctrl+C


ÉTAPE 5 : DÉPLOYER SUR VERCEL
------------------------------
1. Créez un repository GitHub (nommez-le "AUTO-LUXE")
2. Dans le terminal :

    git init
    git add .
    git commit -m "Initial commit"
    git remote add origin https://github.com/VOTRE-USERNAME/AUTO-LUXE.git
    git push -u origin main

3. Allez sur https://vercel.com → "Add New..." → "Project"
4. Importez votre repo "AUTO-LUXE"
5. Dans "Environment Variables", ajoutez :
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
   (Les mêmes valeurs que dans votre .env)
6. Cliquez "Deploy"
7. Attendez 2 minutes

🎉 VOTRE SITE EST EN LIGNE !
